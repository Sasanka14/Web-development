Lecture 1: Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to control the layout, colors, fonts, and overall appearance of HTML web pages.

Why CSS is Used

  • Separates content from design - keeps HTML clean
  • Improves website appearance - better UI/UX
  • Enables consistency - same style across multiple pages
  • Reduces code duplication - easier maintenance
  • Supports responsive design - works on all screen sizes

History of CSS

  • CSS1 (1996) - basic styling like fonts and colors
  • CSS2 (1998) - positioning and media types
  • CSS3 (2005-present) - modular CSS with animations and layouts

Types of CSS

  • Inline CSS - styles applied directly to an element
  • Internal CSS - styles written inside the <style> tag
  • External CSS - styles written in a separate .css file

CSS Syntax

selector {
  property: value;
}

Example of CSS

p {
  color: blue;
  font-size: 16px;
}

Advantages of CSS

  • Improves page loading speed - less HTML code
  • Easy to maintain - change styles in one place
  • Reusable styles - apply same design everywhere
  • Better accessibility - clean structure

Disadvantages of CSS

  • Browser compatibility issues
  • Can become complex in large projects
  • Different browsers may render styles differently

CSS Versions Overview

Version Description
CSS1 Basic styling features
CSS2 Positioning and media support
CSS3 Advanced layouts, animations, responsiveness

Additional Important Points

  • CSS is not a programming language - it does not contain logic
  • CSS works with HTML - HTML provides structure
  • CSS enhances user experience - better visuals
  • CSS is essential for modern web development
