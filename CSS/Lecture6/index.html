Task 7: Inline vs Block Elements

Q1. Which properties behave differently?
-> Inline elements ignore vertical margins and do not expand width like block elements.
Q2. Why does inline padding affect layout but inline margin behaves inconsistently?
-> Padding increases visible and clickable area, while margin doesn't reliably push inline neighbors.
Q3. Convert span to inline-block and observe changes.
-> Inline-block enables full margin and padding behavior like block elements.
