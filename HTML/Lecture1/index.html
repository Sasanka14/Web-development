Lecture 1: Introduction to HTML

History of HTML

  • HTML was invented by Tim Berners-Lee - to enable document sharing over the Internet
  • Developed in 1991 at CERN - for scientific communication
  • Created to share linked documents - using hypertext concept
  • Became the foundation of the World Wide Web

What is HTML?

HTML (HyperText Markup Language) is a standard markup language used to structure web content so browsers can display text, images, and links properly.

What is HTML5?

HTML5 is the modern version of HTML designed to build fast, interactive, and multimedia-rich web applications without external plugins.

Why HTML5 Was Introduced

  • To eliminate dependency on plugins like Flash
  • To support audio and video natively in browsers
  • To improve performance on mobile devices
  • To provide meaningful semantic structure
  • To support modern web application features

What is Flash?

Flash is a proprietary multimedia platform by Adobe used to create animations and interactive content through browser plugins.

Flash vs HTML5

Flash HTML5
Requires external plugin Runs directly in browser
Security vulnerabilities More secure open standard
Not supported on most mobiles Mobile-friendly by design
Proprietary technology Open and standardized
Discontinued in 2020 Actively maintained

Advantages of HTML5

  • Built-in audio and video - no plugins required
  • Semantic tags - better SEO and accessibility
  • Improved performance - faster page loading
  • Offline storage - supports web apps without internet
  • Cross-device compatibility - works on all platforms

Disadvantages of HTML5

  • Older browsers may not fully support features
  • Different browser implementations can cause issues
  • Advanced graphics still require external libraries
  • Improper API use can introduce security risks

Additional Important Points

  • HTML is not a programming language - it cannot perform logic or calculations
  • HTML works with CSS - CSS controls layout, colors, and design
  • HTML works with JavaScript - JavaScript adds behavior and interactivity
  • HTML5 improves SEO - semantic tags help search engines understand content
  • HTML5 is the web foundation - all modern websites are built on it
